# HThompson

[![Project Tracker](https://img.shields.io/badge/repo%20status-Project%20Tracker-lightgrey)](https://hthompson.dev/project-tracker#project-611022081)

A modern personal website serving as my digital hub and online presence. Built with Next.js, it features live GitHub project tracking, centralized access to hosted services, and website policy documentation.

**Live Site:** [hthompson.dev](https://hthompson.dev)

## Tech Stack

- [Next.js](https://nextjs.org/) (App Router)
- [React](https://react.dev/)
- [TypeScript](https://www.typescriptlang.org/)
- [ESLint](https://eslint.org/)
- [Prettier](https://prettier.io/)
- [pnpm](https://pnpm.io/)

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v22+ recommended)
- [pnpm](https://pnpm.io/) (10.14.0+)

### Installation

```bash
git clone https://github.com/StrangeRanger/HThompson.git
cd HThompson
pnpm install
```

## Development

```bash
pnpm dev
pnpm build
pnpm start
pnpm lint
pnpm lint:fix
pnpm format
```
